# 🍽️ Restaurant Management System (ASP.NET Core MVC)

A full-stack Restaurant Management System built using ASP.NET Core MVC.  
This application allows users to browse menu items, add them to a cart, and place orders, while providing admin functionalities to manage restaurant data efficiently.

---

## 🚀 Features

- 👤 User Authentication & Authorization (Login/Register)
- 🍔 Browse Restaurant Menu
- 🛒 Add to Cart & Manage Orders
- 📦 Order Placement System
- 🧑‍💼 Admin Panel for Managing Menu Items
- 🔄 Full CRUD Operations (Create, Read, Update, Delete)
- 🗄️ Database Integration with Entity Framework

---

## 🛠️ Technologies Used

- ASP.NET Core MVC  
- C#  
- Entity Framework Core (Code First)  
- SQL Server  
- Razor Views  
- HTML, CSS, Bootstrap  

---

## 🏗️ Project Architecture

This project follows the **MVC (Model-View-Controller)** architecture:

- **Models** → Handle data and database structure  
- **Views** → UI components using Razor syntax  
- **Controllers** → Business logic and request handling
